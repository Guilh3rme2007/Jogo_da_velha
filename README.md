# Jogo da Velha em C

##

nesse jogo o jogador compete contra a maquina 
(para jogar basta colocar o numero da linha e o da coluna para que o sistema consiga representar a localização)
o jogo é uma matriz 3x3
   |   |
---|---|---
   |   |
---|---|---
   |   | 
      
Jogo feito na linguagem C, com o intuito de desenvolver os conhecimentos em matriz
